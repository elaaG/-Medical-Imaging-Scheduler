# Medical Imaging Scheduler 

A professional scheduling application for medical imaging departments with Gurobi optimization.

## Features
- Multi-machine scheduling (MRI, Scanner, etc.)
- Staff constraints and resource allocation
- Setup time constraints between procedures
- Priority-based optimization
- Interactive Gantt chart visualization
- JSON import/export
- PDF report generation
